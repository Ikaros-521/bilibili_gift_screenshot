# 前言

程序名：B站送礼截图  
功能：用于哔哩哔哩直播时，用户送礼物时触发截图，可自定义截图区域  

# 开发环境

python：3.10.10  
操作系统：win11  

# 使用

## 安装依赖

`pip install requirements.txt`  

## 运行webui

`python webui.py`，启动后自动打开浏览器访问webui页面，完成配置后点击`保存配置`，再`一键运行`即可。
